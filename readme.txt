Resolvi praticar um pouco de POO com Java e acabei me envolvendo com o tema, então mesmo sem ter concluído totalmente o sistema em java, senti vontade de fazer uma interface em site sobre (que acabou resultando em um código mais esforçado e robusto). Então trago o código do Site pronto, mesmo sendo um pequeno projeto, e o esboço que é  o programa Java. 

O sistema em si consiste apenas em uma consulta de veículos com algumas funcionalidades. Esse sistema pode evoluir para um gerenciamento de estacionamento de um estabelecimento através da conexão de banco de dados (que estou preparando). Mas, são planos futuros.

Sistemas: Bootstrap, CSS, HTML e Javascript.

Funcionalidades: 
- Permite adicionar veículos à lista com informações simples necessárias: ano e nome;
- O nome do automóvel tem uma validação que evita que o mesmo veículo seja registrado duas vezes (sofrerá mudanças);
- Os veículos são ordenados por ano (recente->antigo);
- É possível tanto resetar toda a lista quanto excluir apenas um veículo;
- Os dados registrados não se perdem mesmo ao recarregar a página;
- Tem alguns outros sistemas de validação que não permite que seja inserido valores vazios, e mesmo que o usuário tente, o programa não quebra ao inserir outros valores, doutra forma envia um alerta;
- Visual simples e intuitivo;
- Permite a pesquisa do nome do carro na lista caso a mesma seja extensa;
- Equipado com 9 funções;
